Execute a server-side Python script for maintenance or diagnostics. Provide the full filesystem path to a .py file.

API Reference

POST /upload - Upload a file (multipart form, field: file)
GET  /uploads/<filename> - Retrieve an uploaded file
GET  /admin/execute?file=<path> - Run a Python script on the server
